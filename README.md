# vue-ncli
用nginx作为服务器开发vue，同时兼有实时打包，hot-loader，热加载等功能


## 做脚手架的目的
```
最近开始用vue写一个后台管理系统，前后端分离，服务端用java，前端直接调用接口进行渲染。  此间会就产生一个跨域的问题，那么前端资源就需要用nginx作为静态资源服务器，解决跨域的问题。
```

## 用nginx作为静态资源服务的思考
- 想用webpack的hot-reload怎么办
- 开发中错误定位，怎么配置sourceMap
- 实时打包，这个很好处理
